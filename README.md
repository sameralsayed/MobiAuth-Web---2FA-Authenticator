# 🌟 MobiAuth Web - 2FA Authenticator

**A sleek, fully functional browser-based 2FA authenticator that simulates the official MobiAuth Android app (Authenticator App - 2FA Auth).**

Generate secure TOTP & HOTP codes, store them in an encrypted vault, import via QR code or manual entry, and enjoy cloud backup & recovery — all running 100% in your browser with zero installation.

## ✨ Features
- 🔐 **Encrypted Vault** with 4-digit PIN protection  
- 📱 **Live TOTP Generator** – codes refresh every 30 seconds with countdown  
- 📲 **QR Code Scanner Simulation** – one-click add from QR  
- ⌨️ **Manual Token Entry** – support for any service  
- ☁️ **Cloud Backup & Restore** – simulated encrypted backup  
- 📋 **One-click Copy** – copy codes instantly  
- 🌐 **Offline Ready** – works without internet  
- 📱 **100% Responsive** – perfect on mobile, tablet & desktop  
- 🎨 **Modern Dark Security UI** with smooth animations  
- 🔄 **Real-time Progress Circles** for each token  
- 🌍 **SEO Optimized** – rich meta tags and keywords
